Projects

MiniPlay  

MiniPlay is a Chrome extension that adds a miniplayer popup and global keyboard shortcuts to popular streaming music players like Spotify and Google Play Music. It also provides Last.fm scrobbling and rich notifications.

Mobile Night Vision

An iPhone app implementing contrast limited adaptive histogram equalization (CLAHE) to improve contrast in low-light videos. Depends on OpenCV, but only for the cv::Mat structure. Runs at 10ms per frame on 720p video on an iPhone 6S, so it is suitable for augmented reality applications.

espresso

Espresso is a partial implementation of a deep learning framework in the Halide image proessing language compatible with Caffe configuration files.

threadpool

A lightweight header-only threadpool library for C++11. Offers more control than std::async while remaining simpler than manual thread management.
